# Bookings and Reservations

This is the repository for my bookings and reservations project.

- Uses Go version 1.25.3
- Uses [chi package](https://github.com/go-chi/chi) for routing
- Uses [Alexedwards scs package](https://github.com/alexedwards/scs/v2) for session management
- Uses [nosurf](https://github.com/justinas/nosurf) for CSRF protection
